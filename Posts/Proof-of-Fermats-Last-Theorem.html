Proof of Fermat's Last Theorem with Trigonometry

Posted by İdil Doğa Türkmen on April 13, 2022

Introduction

Pierre de Fermat (1601-1665) was a lawyer and was engaged in mathematics as a hobby. About 1637, he explained the so-called "Fermat's Last Theorem" in the now-lost copy of Bachet's translation of Diophantus' Arithmetica:

For integer n > 2 the equation:

an + bn = cn
cannot be solved with positive integers a, b, and c.

He argued that a positive expression cannot be found for the values ​​of a, b, and c in equation, provided that n is greater than 2.

“I have found remarkable proof of this fact, but there is not enough space in the margin of the book to write it down.” (Fermat)

Fermat never published a proof of this statement. It has come to be known as Fermat's Last Theorem because it is not his last work, but the last statement in Fermat's work that must be proved or independently verified posthumously. The theorem was proved by Sir Andrew John Wiles in 1993. Wiles' proof spans over 100 pages and often includes special symbols and notations of group theory, algebraic geometry, commutative algebra, and Galois theory. A simplified trigonometry proof will be presented in this article.

Proof

1. [∀ (a; b; c; n > 2) positive natural number]

In the first step, the prerequisites are specified so that the result of the aforementioned formula cannot be provided by any positive number.

2. an + bn = cn ⇒ (c cosθ)n + (c sinθ)n = cn
(a = c cosθ & b = c sinθ)

In this operation, the expression a is accepted as (a number 'c' multiplied by cosθ. Cosθ can take any value between 0 and 1. This value is multiplied by the positive natural number 'c', which will make a positive natural number. The same is true for 'b'. The reason for multiplying 'b' by sinθ instead of cosθ is that the rule cos2θ + sin2θ = 1 will be used later in the proof.

3. cncosnθ + cnsinnθ = cn or cncosnθ + cnsinnθ ≠ cn

In this step, the expression obtained in step 2 is rewritten and the exponent n is distributed over both c, cos and sin. After the distribution, two different negative statements were written for the situations where this equation is satisfied or not.

4. cosnθ + sinnθ = 1 ↔ n = 2 ('↔' stands for 'if and only if')

In this step, when the number 2 is written instead of n, the expression in step 3 will turn into cos2θ+sin2θ = 1 so the expression in step 3 is converted to cn : cn(cos2θ+ sin2θ) = cn. Since the expression in parentheses is also equal to 1, cn = cn and the equation is satisfied.

5. cosnθ + sinnθ ≠ 1 ↔ n ≥ 3 ('↔' stands for 'if and only if')

In the last step, when n ≥ 3 the expression cosn≥3θ+ sinn≥3θ ≠ 1 occurs and is not satisfied. In this proof, the opposite of the theorem was accepted to be true and it was falsified. Therefore, Fermat’s last theorem was proved to be true.
